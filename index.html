01 About
Why You Shoul Learn Programming?

With the potential for a six-figure salary, unlimited PTO, and the flexibility to work from anywhere, learning programming is a smart choice. Programming is increasingly integral to all industries, making it a valuable skill to have.

Who We Are?

Syntax Jo is a team from Al-Hussein Bin Talal University dedicated to developing talented and professional developers in Jordan. Every year, Syntax Jo offers a variety of courses in fields such as web development, Android development, UX/UI, and more.

How I Will Learn?

At Syntax Bootcamp, you'll start by attending online courses to gain foundational knowledge. You'll then participate in small groups to practice and work on real-world projects throughout your learning journey.

For students at Al-Hussein Bin Talal University, you'll have the opportunity to attend on-site practice labs on campus. Students from other universities can participate in online simulations of these labs.

03 Bootcamp Roadmap
HTML

Learn to create the structure and content of web pages using HTML, including elements for text, images, links, and more.

CSS

Gain skills in styling and designing web pages using CSS, covering layout, typography, colors, and responsive design principles.

Bootstrap

Master the use of Bootstrap to rapidly prototype and build responsive, mobile-first websites with pre-built components and utilities.

Versions Control System (Git)

Learn how to effectively use Git to track changes in code, collaborate with teams, and manage project versions efficiently.

JavaScript

Acquire proficiency in JavaScript to create interactive web pages, handle events, manipulate the DOM, and work with APIs for dynamic content.

APIs

Understand how to integrate and work with APIs to fetch data, perform CRUD operations, and enhance web applications with external functionalities.

Angular

Learn Angular to build dynamic web applications using a component-based architecture. Master concepts like modules, services, and reactive programming to create scalable single-page applications (SPAs).

MySQL

Master MySQL for managing relational databases, learning SQL queries, database design, and interacting with databases from web applications.

Node.js

Master Node.js for building fast, scalable server-side applications. Understand its event-driven, non-blocking I/O model and learn to create RESTful APIs, integrate with databases, and handle asynchronous operations.
